## Como usar

Para utilizar os icones basta importa o componente Icons passando o icone desejado

Exemplo

```javascript
import { SearchIcon } from './components/atoms/Icons';

const ButtonSerach = () => {
  return (
    <button>
      <SearchIcon/>
    </button>
  );
};

```
